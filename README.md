# Equipment & Material Tracking

Static web app for tracking equipment and material logs by project.

## GitHub Pages

This folder is ready to publish from the repository root with GitHub Pages.

Recommended Pages settings:

- Source: Deploy from a branch
- Branch: `main`
- Folder: `/ (root)`

## Data Storage

The app currently stores user-entered project data, log edits, imports, archived projects, admin lists, notes, and settings in each user's browser local storage.

For shared multi-user updates, add a hosted database such as Supabase or Firebase.
